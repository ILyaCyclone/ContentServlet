Content Servlet

One of the following parameters is required:
id_wm 		id of web_metaterm.
alias_wm	alias of web_metaterm.
id_vf		id of file's version.
id_fe		id of person for their photo request.
id_fpa		id of photo album.

Other parameters:
cd		Content-Disposition. Valid values: 1 - download, 2 - inline.
ct		Content Type. Valid values: 1 - returns html.
SWidth		width of requested image in pixels. Example: SWidth=716.
SHeight		height of requested image in pixels. Example: SHeight=500.
quality	 	quality of requested image. Valid values: from 0 to 100. Example: quality=80.
no_cache	Cache-Control. Valid values: any int value - max-age=int value, null - no-cache.

After .../content/ as a part of url can be added filename. Example: .../content/image.jpg&id_wm=int value
